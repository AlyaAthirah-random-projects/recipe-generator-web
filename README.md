# recipe-generator-web

AI Recipe Generator
=====================

This repository is a simple project that utilizes both locally hosted LLMs and OpenAI GPT to generate recipes.

## Requirements
- Powerful enough PC to run Llama3-8b Q4
- AnythingLLM
- Ollama/LMStudio
- PostgreSQL database
- OpenAI api key
